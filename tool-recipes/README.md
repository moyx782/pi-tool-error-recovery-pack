# Tool Recipes

This folder stores verified fixes for tool usage errors.

Use this folder only when:

- a tool is about to be used and there may be known setup issues
- a tool call fails
- a command fails
- an extension / MCP / browser / package reports an error

Only save verified successful fixes.

Do not save:

- passwords
- tokens
- cookies
- API keys
- private credentials
- payment information
- private account data
